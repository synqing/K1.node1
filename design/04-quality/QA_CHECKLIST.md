# UI QA Checklist

Visual:
- Consistent spacing, typography, and color tokens.
- States render correctly (hover, focus, active, disabled, error).

Interaction:
- Keyboard navigation with visible focus rings.
- Motion respects reduced-motion preferences.

Accessibility:
- WCAG AA contrast.
- ARIA roles/labels present where required.
- Screen reader alignment for live regions and status updates.

Performance:
- Component render cost stays within budget.
- No jank in critical interactions.

Documentation:
- Metadata present with version and links.
- Changelog updated for notable changes.
